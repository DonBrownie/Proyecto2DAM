# Digitalización de Hotel

## Descripción
Este proyecto tiene como objetivo **digitalizar la gestión de un hotel**, incluyendo:

- Gestión de empleados.
- Control de horarios y fichajes.
- Generación de nóminas.
- Administración de reservas y servicios (opcional).

El proyecto cuenta con dos aplicaciones:

1. **Aplicación de escritorio:** Implementada en **JavaFX**.
2. **Aplicación móvil:** Implementada en **Kotlin** para **Android**.

## Tecnologías utilizadas
- **Java 17+** y **JavaFX** (App de escritorio)
- **Kotlin** y **Android SDK** (App móvil)
- **IntelliJ IDEA** como IDE
- **PostgreSQL** para base de datos
- Control de versiones: **Git / GitHub**
